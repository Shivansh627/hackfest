# Yatra Sahayak

Welcome to *Yatra Sahayak*, your travel planning assistant! This website helps you plan your trips by providing popular destinations and a travel planner feature.

## Features

- *Travel Planner*: Enter your destination and the number of days to plan your trip.
- *Popular Destinations*: Explore popular places to visit in India.
- *Contact Form*: Reach out for inquiries or bookings.

## Technologies Used

- HTML
- CSS
- JavaScript
